SYSTEM BREAKDOWN

This system generates 3D tree-like structures using L-system rule expansion (formal grammar) and applies rotations/transforms to produce branching patterns. The result supports rapid variation for procedural scenes.

L-systems start from an axiom string and repeatedly apply rewrite rules for a chosen number of iterations. The expanded output is interpreted as drawing/branching commands, creating a tree structure with controllable complexity.

Iteration count, branch angle, segment length, randomness, thickness scaling, and rule sets. These parameters allow the generator to produce many tree silhouettes quickly.
